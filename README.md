产品定位
一句话描述：面向乡村小学教师的AI教案智能生成工具，帮助教师快速生成符合课程标准、适配乡村教学实际的高质量教案。

详细说明：

本产品是一款专为乡村小学教师设计的AI辅助备课工具。乡村教师普遍面临教学任务繁重、教研资源匮乏、专业发展支持不足等困境。
本产品通过大语言模型（LLM）技术，结合国家课程标准与乡村教学实际，为教师提供一键式教案生成服务。教师只需输入学科、年级、课题等基本信息，即可在30秒内获得一份结构完整、内容专业、贴合乡村教学场景的教案。

产品核心价值在于：

降低备课门槛：无需专业教学设计知识，AI自动完成教学目标设定、重难点分析、教学过程编排等复杂工作
提升教案质量：基于课程标准与教学大纲，确保教案的规范性和专业性
适配乡村实际：内置乡村教学场景适配策略，提供乡土化案例和留守学生关怀建议
节省备课时间：将教师备课时间从平均2-3小时缩短至15-30分钟
